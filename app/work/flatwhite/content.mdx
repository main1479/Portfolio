<CaseBlock num="01" label="Overview" heading="What it is">
  Flatwhite is a **short-term rentals and property-management company** in Oradea, Romania. The site serves two audiences at once — travellers looking to book a stay, and property owners considering a managed-rental partner — and routes each to the right flow from the homepage.
</CaseBlock>

<CaseBlock num="02" label="My role" heading="What I did">
  **Frontend developer.** Handover was an Adobe XD design and a written brief. My job was to turn it into a production site the client could maintain after launch.

  - Translating XD comps into HTML, SCSS, and vanilla JavaScript
  - Dual-audience navigation — *Călători* (travellers) and *Investitori* (investors), each with its own flow
  - Responsive layout from mobile up
  - Cross-browser QA before handover
</CaseBlock>

<CaseBlock num="03" label="The brief" heading="What they wanted">
  A site that did two jobs without confusing either audience — a clear booking funnel for guests, and a separate path for property owners considering the management service.

  The build needed to ship, look right at every breakpoint, and stay maintainable by a small team afterwards. No over-engineering for systems that weren't going to be built on top of it.
</CaseBlock>

<CaseBlock num="04" label="What I built" heading="The approach">
  Straightforward frontend work for a small-team handover — focused on getting the two user flows right and keeping the markup readable.

  - **Mobile-first SCSS** with a flat token layer for colour, spacing, and type
  - **Responsive layout** built with CSS grid and flexbox
  - **Component reuse** where the design genuinely repeated — pragmatic, not a top-down architecture
  - **Vanilla JavaScript** for navigation, FAQs, and the contact surfaces
  - **Adobe XD handoff** — pixel-checked across the breakpoints that mattered
</CaseBlock>

<CaseBlock num="05" label="Stack" heading="Tools used">
  - HTML
  - SCSS (Dart Sass)
  - Vanilla JavaScript
  - Adobe XD (handoff)
  - Git
</CaseBlock>

<CaseBlock num="06" label="Outcome" heading="Live and serving">
  The site is live at <a href="https://flatwhite.ro" target="_blank" rel="noreferrer noopener">flatwhite.ro</a>, running both the rentals booking flow and the property-management funnel for the company's two audiences.
</CaseBlock>
